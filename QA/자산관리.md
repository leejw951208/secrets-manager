# QA — 자산 관리

> 일자: ____ · 환경: 로컬/배포 · 뷰포트: 모바일/데스크탑

## 기능
| 항목 | 확인 내용 | 결과 | 비고 |
|---|---|---|---|
| 진입 | 하단 "자산" 탭 → 대시보드 표시 | | |
| 수입 설정 | 수입 카드 → 바텀시트 입력 → 저장 → 남은 돈 반영 | | |
| 지출 추가(일반) | 금액·항목·카테고리·결제방법·날짜 입력 → 저장 → 반영 | | |
| 지출 추가(고정) | "고정 지출" ON 저장 → 당월 생성 + "고정" 배지 | | |
| 남은 돈 | 수입 − 지출 = 남은 돈, 진행 바 | | |
| 카테고리별 | 카테고리 합계·비율·색 표시 | | |
| 지출 달력 | 일자별 합계 표시, 날짜 탭 → 선택일 상세 | | |
| 선택일 상세 | 그 날 지출 목록·금액, 항목 탭 → 수정 | | |
| 고정 자동 생성 | 다음 달 이동 시 고정 지출 자동 생성(중복 없음) | | |
| 수정 | 금액·항목 등 변경 후 저장 → 반영 | | |
| 삭제 | "이 지출 삭제" / 고정은 "고정 지출 해지" 선택 | | |

## 레이아웃 (모바일·데스크탑 각각)
| 항목 | 확인 내용 | 모바일 | 데스크탑 |
|---|---|---|---|
| 수입 바텀시트 | 입력·저장 버튼이 뷰포트 안에 보임(스크롤 불필요) ★데스크탑 주의 | | |
| 지출 폼 | 금액 큰 입력·칩·날짜·버튼 잘림 없음 | | |
| 대시보드 | 카드·달력 겹침/넘침 없음 | | |
| 달력 셀 | 금액 텍스트 셀 밖으로 안 넘침 | | |

## 상태
| 항목 | 확인 내용 | 결과 |
|---|---|---|
| 로딩 | 로딩 표시 | |
| 빈 | 수입·지출 없음 시 표시(남은 돈/예산 0) | |
| 에러 | 조회 실패 안내 | |
