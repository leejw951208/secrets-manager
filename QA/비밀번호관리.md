# QA — 비밀번호 관리(보관함)

> 일자: ____ · 환경: 로컬/배포 · 뷰포트: 모바일/데스크탑

## 기능
| 항목 | 확인 내용 | 결과 | 비고 |
|---|---|---|---|
| 진입 | 잠금해제 후 보관함 목록 표시 | | |
| 목록 | 항목·개수 표시, 빈 상태 안내 | | |
| 검색 | 제목 일부로 필터, 없는 검색어 시 "검색 결과가 없어요" | | |
| 검색 입력 | 빠른 타이핑·공백 입력 시 글자 사라짐/딜레이 없음 | | |
| 신규 추가 | 제목 입력 → 다음 → "저장 전 확인" → 암호화하여 저장 → 목록 반영 | | |
| 필드 구성 | 필드 추가/추천 칩/순서이동/삭제 동작 | | |
| 민감 토글 | 폼에서 🔒/👁 전환, 리뷰에서 민감 필드 마스킹 | | |
| 상세 | 평문 필드 노출, 민감 필드 마스킹 → "표시"로 값 노출 | | |
| 복사 | 복사 후 카운트다운, 30초 뒤 클립보드 자동 클리어 | | |
| 필드 삭제 | 개별 필드 "삭제" → 한 번에 제거(나머지 유지) | | |
| 수정 | 제목·필드 변경 후 저장 → 반영 | | |
| 전체 삭제 | 상세 "삭제" → 확인 → 목록에서 제거 | | |
| 백업·복원 | 내보내기 다운로드, 복원 충돌 모드 선택 | | |

## 레이아웃 (모바일·데스크탑 각각)
| 항목 | 확인 내용 | 모바일 | 데스크탑 |
|---|---|---|---|
| 폼 | 입력·버튼이 화면 안에 보이고 잘리지 않음 | | |
| 확인 다이얼로그 | 삭제 확인창과 버튼이 뷰포트 안에 보임(스크롤 불필요) | | |
| 상세 | 긴 값/많은 필드에서 겹침·넘침 없음 | | |
| 하단 탭바/FAB | 가려짐 없이 동작 | | |

## 상태
| 항목 | 확인 내용 | 결과 |
|---|---|---|
| 로딩 | 스켈레톤/로딩 표시 | |
| 에러 | 네트워크·복호화 실패 시 안내 | |
| 빈 | 항목 없음 안내 | |
