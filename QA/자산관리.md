## 4. 카드 결제월(익월 반영)

| #   | 항목        | 확인 내용                                                            | 결과 | 비고 |
| --- | ----------- | -------------------------------------------------------------------- | ---- | ---- |
| 4.1 | 카드 익월   | 6월에 **카드** 지출 입력 → 6월 화면 남은 돈/달력에서 **빠지지 않음** |      |      |
| 4.2 | 익월 등장   | 같은 카드 지출이 **7월 화면**(결제월)에 등장                         |      |      |

| 4.5 | 구매일 부제 | 선택일 상세에서 카드 건에 "M/D 구매" 부제 표시                       |      |      |
| 4.6 | 합계 일치   | 지출 합계 = 남은 돈 차감 금액 일치                                   |      |      |
| 4.7 | 말일 클램프 | 카드 1/31 구매 → 2월 결제는 2/28(말일)로 표시                        |      |      |

## 5. 고정 지출 — 생성·시작월·개월 수

| #    | 항목             | 확인 내용                                                               | 결과 | 비고 |
| ---- | ---------------- | ----------------------------------------------------------------------- | ---- | ---- |
| 5.1  | 토글             | 지출 추가 폼 "고정 지출" 토글(신규만)                                   |      |      |
| 5.2  | 개월 수 입력     | 고정 ON 시 "개월 수 · 선택" 입력 나타남(숫자만, 비우면 무기한)          |      |      |
| 5.3  | 생성             | 고정 ON 저장 → 당월 인스턴스 + "고정" 배지                              |      |      |
| 5.4  | 시작월           | 설정한 달부터 생성. **이전 달로 이동 시 그 달엔 생성 안 됨**(소급 없음) |      |      |
| 5.5  | 카드 익월 시작   | **카드** 고정 6월 설정 → 6월 화면엔 없고 **7월 화면부터** 등장(7·8·9…)  |      |      |
| 5.6  | 비카드 당월 시작 | 현금·자동이체 고정 6월 설정 → 6월부터 등장                              |      |      |
| 5.7  | 개월 수 종료     | 개월 수 3 + 6월 시작(비카드) → 6·7·8월 생성, **9월엔 미생성**           |      |      |
| 5.8  | 개월 수 카드     | 개월 수 3 + 6월 시작(카드) → 결제월 7·8·9월 등장, **10월엔 없음**       |      |      |
| 5.9  | 무기한           | 개월 수 비우고 생성 → 먼 미래 달에도 계속 생성                          |      |      |
| 5.10 | 자동 생성 멱등   | 같은 달 재진입·새로고침에 중복 인스턴스 안 생김                         |      |      |

## 6. 고정 지출 — 해제·삭제

| #   | 항목           | 확인 내용                                                                | 결과 | 비고 |
| --- | -------------- | ------------------------------------------------------------------------ | ---- | ---- |
| 6.1 | 액션 노출      | 고정 인스턴스 수정 화면에 "고정 해제"(보조) + "삭제"(빨강) 둘 다 표시    |      |      |
| 6.2 | 해제           | "고정 해제" → 이후 달 자동 생성 **중단**, 기존·이번 달 인스턴스 **유지** |      |      |
| 6.3 | 해제 후 미생성 | 해제 후 다음 달 이동 시 해당 고정 인스턴스 생성 안 됨                    |      |      |
| 6.4 | 삭제 선택      | "삭제" → "이 고정 전체 삭제 / 이번 달만 삭제 / 취소" 시트                |      |      |
| 6.5 | 전체 삭제      | "이 고정 전체 삭제" → 모든 달 인스턴스 + 규칙 제거                       |      |      |
| 6.6 | 이번 달만 삭제 | "이번 달만 삭제" → 그 건만 사라짐                                        |      |      |
| 6.7 | 재생성 안 됨   | 이번 달만 삭제 후 **새로고침·재진입에도 되살아나지 않음** ★핵심          |      |      |
| 6.8 | 일반은 단일    | 일반 지출은 "이 지출 삭제" 단일 버튼(선택지 없음)                        |      |      |
