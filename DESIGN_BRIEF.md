# Secrets Manager — 화면 요구사항 브리프

> 클로드 디자인 입력용. 비주얼 디자인(색·타입·모션·아이덴티티)은 규정하지 않고, 무엇을 만들지(기능·구조·흐름)만 명세한다.

## 1. 제품
1인용 암호화 비밀번호 보관함. 은행·카드·증권·쇼핑 등 사이트별 비밀번호를 한곳에 암호화 보관하고, 잠깐 열람한 뒤 다시 잠근다. 공개 호스팅(Cloudflare Access가 경계 방어)이며 본인 한 명만 사용한다. 모바일 우선 PWA.

## 2. 사용자
보안에 민감한 단일 소유자. 필요한 한 줄을 빠르게 꺼내 복사하고 다시 잠그는 속도가 핵심.

## 3. 인증 모델 — passkey + WebAuthn PRF + 복구코드
PIN·마스터 비밀번호 없음. passkey로 인증하고, WebAuthn **PRF**로 암호화 키를 도출한다(키는 서버에 저장하지 않음). 기기 분실 대비로 고엔트로피 **복구코드**를 키에 추가 래핑한다.

```
Cloudflare Access(경계)
  → passkey 인증(생체/기기) + PRF로 vault 키 도출
     → 비밀번호 목록 · 열람
        → idle N분 후 키만 메모리에서 폐기(재잠금) → passkey 재인증
```

- 최초 — passkey 등록 + 복구코드 1회 발급(반드시 저장하도록 유도)
- 재방문 — passkey 인증 한 번으로 잠금해제
- 분실 — 복구코드로 접근 복구 + 새 passkey 재등록

## 4. 그릴 화면

### 인증
1. **온보딩 / passkey 등록** — 제품 소개, "passkey 만들기" 단일 CTA, 생체/기기 인증 안내, 진행 단계(등록 → 복구코드).
2. **복구코드 발급** — 복구코드 표시, 복사·다운로드, "안전한 곳에 보관" 경고, "저장했습니다" 확인 후에야 완료.
3. **잠금해제(재방문)** — "passkey로 잠금해제" CTA, 인증 대기/실패/재시도, "복구코드로 접근" 대안.
4. **복구코드로 복구** — 코드 입력·검증, 성공 시 새 passkey 재등록 유도.

상태 — 로딩(상태 확인 중), 에러(네트워크/인증기 미지원), rate-limit/backoff 안내.

### 보관함
5. **목록** — 제목, 자동 잠금 카운트다운, 수동 잠그기. 선택적 카테고리(분류 태그) 필터 + 제목 검색. 항목 카드 목록(제목 + 선택 카테고리). 빈 상태 = 첫 항목 추가 유도.
6. **상세** — 제목·메타 / 사용자 정의 필드 목록(이름 + 값, 값은 기본 마스킹·표시·복사, 복사 후 30초 뒤 자동 재마스킹) / 메모 / 액션(수정·삭제).
7. **신규 · 수정 폼** — 사용자가 필드를 직접 구성한다. (1) 제목 입력 → (2) 필드 추가(필드 이름 입력) → (3) 필드별 값 입력 → (4) 확인(저장 전 리뷰). 필드는 추가·삭제·이름 변경·재정렬이 가능하며 최대 20개. 선택적으로 카테고리(분류 태그)와 메모를 단다. 제목만 평문이고 필드 이름·값·메모는 통째로 암호화된다.
8. **필드 추천(작성 도움)** — 자주 쓰는 필드 이름(아이디·비밀번호·계좌번호·핀 등) 추천 목록. 읽기 전용이며 폼에서 빠르게 추가하는 보조 수단이다.
9. **백업 · 복원** — 암호화 export 다운로드 / import(파일 + 충돌 모드 reject·skip·replace).
10. **404 / 미존재 항목** — 한국어 안내 + 복귀.

## 5. 제약
- 접근성 — WCAG AA, 키보드 포커스 가시, aria, 터치 타깃 44px.
- 반응형 — 모바일(≈375) → 데스크탑(≈1280).
- PWA — 오프라인 셸.
- 한국어 UI.
