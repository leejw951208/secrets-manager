# QA 양식

수동·자동(Playwright) QA용 체크리스트. 기능별 양식은 같은 폴더의 파일을 사용한다.

- [비밀번호 관리](./비밀번호관리.md)
- [자산 관리](./자산관리.md)

## 기본 원칙
- **두 종류를 나눠 본다**: 기능(동작하는가) + 레이아웃(잘림·겹침 없이 뷰포트 안에 보이는가).
- **모바일·데스크탑 둘 다** 확인한다(레이아웃 버그는 한쪽에서만 나옴).
- 다이얼로그·바텀시트·하단 버튼은 **뷰포트 안에 실제로 보이는지** 확인한다(클릭만 되는지가 아니라).
- 상태(로딩·에러·빈)도 확인한다.

## 기록 표기
- 결과: ✅ 통과 / ❌ 실패 / ⏭ 해당없음
- 실패는 비고에 재현 방법·환경을 남긴다.
